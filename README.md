API REST de e-commerce desarrollada con C# y ASP.NET Core 8 siguiendo 
arquitectura en capas (Controllers/Services/Repository).

Incluye gestión completa de usuarios con autenticación JWT y hash de 
contraseñas con BCrypt, productos con filtrado dinámico por categoría, 
nombre y rango de precios, paginación y categorías.

Persistencia de datos con Entity Framework Core y SQL Server, aplicando 
el patrón Repository e inyección de dependencias. Documentación automática 
de endpoints con Swagger/OpenAPI.

En desarrollo: gestión de pedidos con control de stock automático, 
sistema de reseñas con validación de compra previa y mejoras de seguridad.
